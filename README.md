# The Combustible Potato (FRC 5113)
*<pre>Born of sparks and starchy flame,
The Combustible Potato bot rose to fame;
With firey circuits and gears aglow,
a legend destined to burn and grow.</pre>*

### Tank Drive
- Basic arm non-pid implementation
- Intake mechanism for the arm
- Custom finite state machines used throughout for reliability

### Teleop Control
- Dual-joystick input
- Deadbands for smooth motion

## Requirements
requirements.txt is included for all packages.

## Test Code

```
robotpy test
```

## Run Simulation

```
robotpy sim
```

## Deploy to Robot

```
robotpy deploy
```
